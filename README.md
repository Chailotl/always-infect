## Usage
It'll automatically work when a villager is killed by an undead.

## Config
```yaml
# Chance for villagers to become zombified
chance: 1.0
```

## Commands
**/alwaysinfect info** - Plugin version

**/alwaysinfect reload** - Reload config (OP)

## Permissions
alwaysinfect.reload (OP)
